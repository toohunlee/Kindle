# Cover Image Setup

## 커버 이미지 설정 방법

Kindle에서 idle time에 표시될 커버 이미지를 설정하려면:

1. 원하는 이미지를 이 폴더(`assets/`)에 저장
2. 파일 이름을 `cover.jpg` 또는 `cover.png`로 변경

## 지원 형식

- JPG/JPEG
- PNG

## 권장 사양

- 해상도: 1600x2560 픽셀 (Kindle Paperwhite 기준)
- 또는 최소 800x1280 픽셀
- 파일 크기: 2MB 이하

## 현재 설정

첨부하신 New York Times 이미지를 `cover.jpg`로 저장하세요.
